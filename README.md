Full featured GameBoy emulator. Let's dance!

```s
$ cargo run -- "mods/Boxes (PD).gb"
```

# Control

```
                    _n_________________
                    |_|_______________|_|
                    |  ,-------------.  |
                    | |  .---------.  | |
                    | |  |         |  | |
                    | |  |         |  | |
                    | |  |         |  | |
                    | |  |         |  | |
                    | |  `---------'  | |
                    | `---------------' |
                    |   _ GAME BOY      |
       Up           | _| |_         ,-. | ----> Z
   Left/Right <---- ||_ O _|   ,-. "._,"|
      Down          |  |_|    "._,"   A | ----> X
                    |    _  _    B      |
                    |   // //           |
                    |  // //    \\\\\\  | ----> Enter/BackSpace
                    |  `  `      \\\\\\ ,
                    |________...______,"
```
